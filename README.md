# 🎲 Dice Game

This is a simple and fun dice game built using JavaScript where two players roll a dice and the player with the higher number wins. It is a beginner-friendly project that demonstrates basic DOM manipulation, random number generation, and game logic.

---

## ✨ Features

* 🎲 Random dice roll (1–6)
* 👥 Two player mode
* 🏆 Displays winner automatically
* 🔄 Refresh to play again
* ⚡ Fast and simple UI

---

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript

---

## 🚀 How to run

1. Clone the repository

```
git clone https://github.com/your-username/dice-game.git
cd dice-game
```

2. Open the project
   👉 Just open `index.html` in your browser

---

## 🎮 How it works

* Each player gets a random number between 1 and 6
* Dice images update based on the number
* The player with the higher number wins
* If both numbers are same → it's a draw
